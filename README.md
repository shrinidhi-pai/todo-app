# 📝 To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

* ➕ Add tasks
* ✏️ Edit tasks
* 🗑️ Delete tasks
* 💾 LocalStorage support
* 📱 Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
